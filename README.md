# Product Feedback Form

This is a simple web application built using **HTML** and **JavaScript**.  
It allows users to submit their feedback about products and displays the submitted data on the same page.

## Features
- Collects user information (name, age, email, job, designation)
- Displays feedback instantly on submission
- Supports both button click and "Enter" key submission
- Built for Coursera hands-on lab (Develop Product Feedback Form)

## Technologies Used
- HTML  
- JavaScript
